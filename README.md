Some simple card games that I made

The games require cards.py to be within the same folder the game

Blackjack is very basic and my first run at the game

Blackjack_money allows you to "bet" against the computer doubling down and split hands

rideTheBus is a cmdline replica of the ride the bus game as seen in schedule 1 not the drinking game

scoreCheck will create its own highscore.json file if it does not exist already if you would like to change the path it exists in this file
